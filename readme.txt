== Description ==

Display reports of number posts published by every Author on your Wordpress installation.

== Basic Features ==

Reports Included in the Plugin:

* Display number post by Author

== Follow Us == 

[github](https://github.com/felixsinggih) 


== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Manage your Author Post Counter plugin from Left Side Admin Dashboard
